## Web-based Railway Ticket Booking Management System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/1b49658d40cf41d48baff9037d7b3133/ghb20250916) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Homepage (About Us, System Introduction)
4. Personal Center (Modify Personal Information, Change Avatar, Change Password)
5. Train Information Query
6. Announcement Information Viewing
7. Personal Data Background (Ticket Booking, Refund, etc.)
8. Train Ticket Favorites
9. Train Ticket Comments

**Administrator Functions:**
1. Login
2. Personal Center
3. Change Password
4. User Management
5. Train Type Management
6. Train Information Management
7. Ticket Booking Management
8. Ticket Refund Management
9. System Management (Announcement Information Management, About Us, System Introduction Settings, Carousel Management)
